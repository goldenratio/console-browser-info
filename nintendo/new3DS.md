# New 3DS

Browser supports HTML5 canvas.

#### Key Events
The following buttons trigger the onkeydown, onkeypress and onkeyup events:

| Keycode        | Button|
| ------------- |:-------------:|
| 13 | A |
| 37 | Left Arrow key |
| 38 | Up Arrow key |
| 39 | Right Arrow key |
| 40 | Down Arrow key |

Touch events are supported on the secondary screen.
