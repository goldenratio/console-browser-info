#PS Vita

Browser supports HTML5 Canvas

#### Key Events
The following buttons trigger the onkeydown, onkeypress and onkeyup events:

| Keycode        | Button|
| ------------- |:-------------:|
|  | X |
|  | Left Arrow key |
|  | Up Arrow key |
|  | Right Arrow key |
|  | Down Arrow key |

Touch events are also supported.
