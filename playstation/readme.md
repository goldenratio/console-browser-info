#PS Vita

#### Key Events
The following buttons trigger the onkeydown, onkeypress  and onkeyup events:

| Keycode        | Button|
| ------------- |:-------------:|
| 37 | Left Arrow key |
| 38 | Up Arrow key |
| 39 | Right Arrow key |
| 40 | Down Arrow key |
| Mouse Click | X |

Other buttons do not trigger key events.
Touch events are supported.
